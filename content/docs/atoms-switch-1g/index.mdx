# Atoms Smart Switch 1G

The Atoms Smart Switch 1G is a professional-grade Wi-Fi smart wall switch built for
reliable local-first operation. It fits standard 86×86mm single-gang wall boxes and
integrates natively with Home Assistant, ESPHome, and Tasmota.

## What's in the Box

- 1× Atoms Smart Switch 1G unit
- 1× Switch plate (white)
- 4× M3 mounting screws
- 1× Quick Start card

## Key Capabilities

| Feature | Details |
|---|---|
| Load switching | Up to 10A / 2200W resistive |
| Connectivity | Wi-Fi 802.11 b/g/n (2.4GHz) |
| Protocol | HTTP REST, MQTT, mDNS |
| MCU | ESP32-C3 |
| Firmware | Atoms v1.x, ESPHome, Tasmota |
| Local control | Yes — works without internet |

## Before You Start

Make sure your installation meets these requirements:

- **Neutral wire available** — The switch requires a neutral (N) connection. Check your wall box before ordering.
- **Single-gang wall box** — Minimum installation depth 35mm.
- **2.4GHz Wi-Fi** — 5GHz is not supported.
- **10A max load** — Do not exceed 10A on the switched circuit.
